# P006-04-01: 動画詳細 (/works/youtube/sample-youtube-1/)

## 基本情報
- 主目的: 投稿した動画の詳細情報を提供し、視聴者に動画の内容や価値を伝える。
- 想定ユーザー: YouTubeチャンネルの視聴者、ポートフォリオ閲覧者、協業相手。
- 想定導線:
  - 主: P006-04（YouTube）の動画一覧カードから遷移する。
  - 副: YouTube 上の動画説明欄や SNS から本ページへ直接遷移する。

## 識別情報（管理画面連携）

| 項目 | 詳細 | 備考 |
|------|------|------|
| slug | URL 用識別子（例: `sample-youtube-1`） | 英小文字・数字・ハイフンのみ。一覧・詳細で共通 |
| youtube_video_id | YouTube 動画 ID（11 文字） | 本編視聴リンク・サムネイル取得に使用 |
| published_at | YouTube 公開日 | P006-04 一覧の「投稿年月日」と同一ソース |
| is_published | 公開 / 下書き | 下書き時は一覧・詳細とも非表示 |

## 表示内容

| 要素名 | 詳細 | 備考 |
|--------|------|------|
| 動画名 | ページ見出し（H1）として表示 | パンくず・OGP タイトルに使用。P006-04 一覧の動画名と同一 |
| 投稿年月日 | YouTube 公開日を表示 | `published_at` を `YYYY年M月D日` 形式等で表示 |
| サムネイル | 投稿動画のサムネイルを表示 | 一覧カードと同一画像を使用。`alt` は動画名を設定。保存方式は案C（YouTube 自動取得 + カスタム画像差し替え）。下記「サムネイル管理」参照 |
| 概要 | 動画の概要を簡潔に記載 | 段落または箇条書き。目安 200〜500 文字 |
| 反省点 | 動画の改善点や次回への課題を記載 | 段落または箇条書き |
| YouTube リンク | 本編視聴への外部リンクを表示 | 新規タブで開く。`rel="noopener noreferrer"` を付与 |
| 関連リンク | 動画に関連するリンクを表示 | リンクが存在する場合のみ表示（下記参照） |
| 一覧へ戻る | P006-04 へのリンクを表示 | `画面遷移図.md` 備考に準拠 |

### 関連リンク表示について

| 要素名 | 詳細 | 備考 |
|--------|------|------|
| ラベル | リンクの表示文言 | 例: 「使用素材」「関連記事」 |
| URL | リンク先 | 外部リンクは新規タブ。YouTube 本編リンクは上記「YouTube リンク」と重複させない |
| 件数 | 0 件以上 | 0 件の場合はセクションごと非表示 |

## サムネイル管理（案C: ハイブリッド）

- 既定は YouTube 自動取得サムネイル（`youtube_video_id` から `https://i.ytimg.com/vi/{id}/hqdefault.jpg` を組み立て）を表示する。
- 必要な動画のみ管理画面でカスタム画像に差し替える（`thumbnail_source = custom` + アップロード画像）。
- 公開 API は解決済みの `thumbnail_url` を返却し、一覧（P006-04）と詳細で同一画像を使用する。
- データ・API 詳細: `01_DOCS/04_API設計/02_詳細設計/テーブル定義/youtube_videos.dbml`、`YouTube動画一覧取得API.md`、`メディアアップロードAPI.md`。

## SEO・アクセシビリティ

- OGP: 動画名・概要の抜粋・サムネイル画像を設定する（`05_SEOとアクセシビリティ方針.md` 参照）。
- 構造化データ: `VideoObject` の実装を検討する（`youtube_video_id` を `embedUrl` 等に利用）。
- サムネイル: `<img>` に `alt` を必須とし、`loading="lazy"` を付与する。
- パンくず: `共通仕様.md` 2.4 の P006-04-01 定義に準拠する。
