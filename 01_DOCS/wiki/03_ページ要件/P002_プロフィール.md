# P002: プロフィール (/about/)

## 基本情報
- 主目的: サイト訪問者に自己紹介する。
- 想定ユーザー: 採用担当者、協業相手、ポートフォリオ閲覧者。
- 想定導線: トップページから遷移する。

## 表示内容

| 要素名    | 詳細                                                     | 備考                                               |
|--------|--------------------------------------------------------|--------------------------------------------------|
| 目次     | ページ内リンクで「自己紹介」「スキルセット」「リンク」へ移動できる目次を配置。                | セクション見出しとアンカーを一致させる。モバイルでは先頭固定ではなく通常配置として可読性を優先。 |
| 自己紹介   | 自己紹介文を表示する                                               |                                       |
| スキルセット | 「フロントエンド」「バックエンド」「デザイン」「その他」のカテゴリでカードを作成し、それぞれのカテゴリ概要を表示する。 | 個別スキルの詳細は P002-01〜04 で表示。詳細ページへのリンクは準備中でない場合のみ表示。 |
| リンク    | 各種SNSやブログへのリンクを表示する                                      | GitHub は必須。SNS・ブログは任意。それぞれのリンク先に合わせたアイコンも表示する。 |

### スキルセットカード表示について
| 要素名    | 詳細                                            | 備考            |
|--------|-----------------------------------------------|---------------|
| アイコン   | 「フロントエンド」「バックエンド」「デザイン」「その他」それぞれを示すアイコンを表示する。 |               |
| 名称     | 「フロントエンド」「バックエンド」「デザイン」「その他」のいずれかを表示する。       |               |
| 準備中バッジ | 詳細ページが準備中の場合、表示する。                            | カード全体の色を薄くし、リンクを非表示とする |
| 概要     | 「フロントエンド」「バックエンド」「デザイン」「その他」のいずれかの詳細説明を表示する。  |               |
| リンク    | 詳細ページへのリンクを表示する。                              |               |
