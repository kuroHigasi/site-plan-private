# P005-01: 資格取得一覧 (/certifications/list/)

## 基本情報
- 主目的: 管理者が取得している資格の一覧を表示する。
- 想定ユーザー: 採用担当者、協業相手、ポートフォリオ閲覧者。
- 想定導線: 資格・学習コンテンツトップから遷移する。


## 表示内容

| 要素名          | 詳細                                        | 備考              |
|--------------|-------------------------------------------|-----------------|
| カテゴリ絞り込み     | 「すべて」「IT」「デザイン」「数学」「言語」「その他」の各カテゴリで絞り込み可能 | 存在しないカテゴリは表示しない |
| 資格一覧         | 取得済みの資格をカード表示して一覧とする                      |                 |

### 資格カード表示について
| 要素名  | 詳細                       | 備考                 |
|------|--------------------------|--------------------|
| アイコン | カテゴリ毎に表示                 |                    |
| 資格名  | 資格の名称を表示                 |                    |
| 取得年月 | 資格を取得した年月を表示             |                    |
| リンク  | 資格を取得した報告をまとめたブログのリンクを表示 | ブログを未作成の場合はリンクを非表示 |
