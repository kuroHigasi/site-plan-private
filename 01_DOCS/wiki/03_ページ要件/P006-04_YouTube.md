# P006-04: YouTube (/works/youtube/)

## 基本情報
- 主目的: 本サイトのYouTubeチャンネル運用状況を紹介し、関連する作品やコンテンツの価値を伝える。
- 想定ユーザー: YouTubeチャンネルの視聴者、ポートフォリオ閲覧者、協業相手。
- 想定導線: トップページ・作品一覧から遷移し、必要に応じて「チャンネル概要」「動画一覧」へ移動する。

## 表示内容

| 要素名     | 詳細                                  | 備考                                               |
|---------|-------------------------------------|--------------------------------------------------|
| 目次      | ページ内リンクで「チャンネル概要」「動画一覧」へ移動できる目次を配置。 | セクション見出しとアンカーを一致させる。モバイルでは先頭固定ではなく通常配置として可読性を優先。 |
| チャンネル概要 | チャンネルの概要を簡潔に記載。                     | データソース: `GET /api/public/youtube-channel.php`（`YouTubeチャンネル取得API.md`） |
| 動画一覧    | 投稿動画の詳細ページへ飛ぶカードを表示。                | 公開済み（`is_published = 1`）の動画のみ表示。データソース: `GET /api/public/youtube-videos.php`（`YouTube動画一覧取得API.md`） |

### 投稿動画カード表示について
| 要素名   | 詳細                      | 備考 |
|-------|-------------------------|----|
| サムネイル | 動画のサムネイルを表示             | P006-04-01 詳細と同一画像。`alt` は動画名 |
| 動画名   | 動画の名称を表示                |    |
| 投稿年月日 | YouTube 公開日を表示           | P006-04-01 の `published_at` と同一ソース |
| リンク   | 動画に関する詳細ページへと遷移するリンクを表示 | `/works/youtube/details?slug={slug}` へ遷移。API レスポンスの `detail_href` をそのまま使用する |
