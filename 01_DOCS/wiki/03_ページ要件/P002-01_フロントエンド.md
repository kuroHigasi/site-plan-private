# P002-01: フロントエンド (/about/frontend/)

## 基本情報
- 主目的: フロントエンド分野のスキルと経験を詳しく伝える。
- 想定ユーザー: 採用担当者、協業相手、ポートフォリオ閲覧者。
- 想定導線: プロフィール（P002）のスキルセットカードから遷移する。

## 表示内容

| 要素名        | 詳細                                           | 備考                                    |
|------------|----------------------------------------------|---------------------------------------|
| カテゴリ概要     | フロントエンド分野の説明を表示する。                           | P002 のスキルセットカード「概要」と内容を整合させる。        |
| 絞り込み     | 「すべて」「言語」「フレームワーク」「スタイリング」「開発ツール・テスト」で表示スキルカードを絞り込み可能とする。                           | 存在しないカテゴリは表示しない。        |
| スキル一覧      | 保有スキルをカード形式で表示する                       |             |
| プロフィールへ戻る | プロフィール（P002）へのリンクを表示する。                      | ページ下部に配置する。                           |

### スキルカード表示について

| 要素名        | 詳細                                           | 備考                                    |
|------------|----------------------------------------------|---------------------------------------|
| アイコン     | 「言語」「フレームワーク」「スタイリング」「開発ツール・テスト」それぞれを示すアイコンを表示する。                          |        |
| 名称     | スキルの名称を表示する。                           |        |
| 概要     | スキルの概要を表示する。                           |        |
| レベル     | 習熟度のレベルをバーと数値（1～5の5段階）で表示する。      | 各段階の定義は本ページ「備考」を参照。        |
| 根拠     | 関連資格や関連作品があればリンクを表示する。      | 資格は P005 系ページ、作品は P006 系ページへの内部リンク。複数ある場合はリスト表示。リンクテキストは資格名または作品名。ない場合は非表示。       |

## 備考
- 参考: `01_DOCS/02_サイト構成/画面遷移図.md`、`01_DOCS/wiki/01_コンセプト定義/05_SEOとアクセシビリティ方針.md`
- 習熟度レベル（1〜5）: 1=入門（学習開始）、2=基礎（教材レベルで理解）、3=個人開発で使用可能、4=実務で継続使用、5=設計・指導が可能
- 表記ルール: 技術名は正式名称で統一し、省略語を使う場合は初出で正式名称を併記する。
- 更新運用: ページ要件更新時は関連する`02_OUTPUT`の最新スクリーンショットとの差分確認を行い、不整合があれば先に要件を修正する。
