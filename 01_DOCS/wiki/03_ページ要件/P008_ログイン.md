# P008: ログイン (/login/)

## 基本情報
- 主目的: 管理者が管理画面にログインする。
- 想定ユーザー: サイト管理者。
- 想定導線: フッターから遷移する。

## 表示内容

| 要素名     | 詳細                     | 備考                               |
|---------|------------------------|----------------------------------|
| ユーザー名   | 管理者のユーザー名を入力するフィールド。   | 必須入力。                            |
| パスワード   | 管理者のパスワードを入力するフィールド。   | 必須入力。                            |
| ログインボタン | 入力内容を送信してログインを実行するボタン。 | ログイン成功時は管理画面へ遷移。失敗時はエラーメッセージを表示。 |

- 参考: `01_DOCS/02_サイト構成/画面遷移図.md`、`01_DOCS/wiki/01_コンセプト定義/05_SEOとアクセシビリティ方針.md`
