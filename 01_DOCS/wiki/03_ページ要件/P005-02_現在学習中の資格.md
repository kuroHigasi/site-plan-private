# P005-02: 現在学習中の資格 (/certifications/current/ )

## 基本情報
- 主目的: 管理者が現在学習中の資格の一覧を表示する。
- 想定ユーザー: 採用担当者、協業相手、ポートフォリオ閲覧者。
- 想定導線: 資格・学習コンテンツトップから遷移する。


## 表示内容

| 要素名          | 詳細                                        | 備考              |
|--------------|-------------------------------------------|-----------------|
| カテゴリ絞り込み     | 「すべて」「IT」「デザイン」「数学」「言語」「その他」の各カテゴリで絞り込み可能 | 存在しないカテゴリは表示しない |
| 中断中資格の非表示トグル | 中断中の資格を非表示にするかどうかを切り替えるトグル                | 初期値: 表示         |
| 資格一覧         | 現在学習中の資格をカード表示して一覧とする                     |                 |

### 資格カード表示について
| 要素名    | 詳細                       | 備考                     |
|--------|--------------------------|------------------------|
| アイコン   | カテゴリを示すアイコンを表示           |                        |
| 中断中バッジ | 中断中の資格を示すバッジを表示          | 中断中の場合はカードの背景色も他より薄くする |
| 資格名   | 資格の名称を表示          |                        |
| 学習開始年月 | 資格の学習を開始した年月を表示          |                        |
| 取得予定年月 | 資格を取得予定の年月を表示            |                        |
| 進捗     | 資格の進捗状況をバーとパーセンテージで表示    |                        |
| 現在学習状況 | 直近で学習している内容を表示           |                        |
