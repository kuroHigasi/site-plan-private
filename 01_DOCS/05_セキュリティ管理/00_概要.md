本リポジトリは公開ブランチであるため、セキュリティに関する下記要件は別リポジトリ（site-plan-security）にまとめるものとする。
- ログインに関するAPIやパスワード管理の仕様・設計
    - API設計、DB設計
- ログイン後に確認できる管理画面の詳細な仕様
    - 画面遷移図、ページ要件
- 管理画面から呼び出す管理 API（`/api/admin/`）の仕様・設計
    - コンテンツ CRUD（changelogs / qualifications / skills / youtube / users 等）の API 仕様、テーブル定義（DBML）

これらの正本は site-plan-security の `01_DOCS/04_API設計/` および `01_DOCS/wiki/04_API設計/` にある。本リポジトリでは公開 API（`/api/public/` 読み取り系）の仕様のみを管理する。
