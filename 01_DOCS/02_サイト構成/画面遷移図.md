# 画面遷移図

ポートフォリオサイトの画面遷移図叩き台です（Mermaid 記法）。

---

## 画面遷移図

```mermaid
flowchart TD
    TOP["🏠 トップページ (/)"]
    ABOUT["👤 プロフィール (/about)"]
    WORKS["🗂️ 作品一覧 (/works)"]
    GAMES["🎮 ゲーム一覧 (/works/games)"]
    GAME_DETAIL["🎮 ゲーム詳細 (/works/games/:id)"]
    UI["🖼️ UI一覧 (/works/ui)"]
    UI_DETAIL["🖼️ UI詳細 (/works/ui/:id)"]
    CERT["📋 資格・学習コンテンツ (/certifications)"]
    CERT_LIST["🏆 取得資格一覧 (/certifications/list)"]
    CERT_CONTENTS["📚 学習コンテンツ一覧 (/certifications/contents)"]
    CERT_DETAIL["📄 学習コンテンツ詳細 (/certifications/contents/:id)"]
    CONTACT["✉️ お問い合わせ (/contact)"]

    TOP --> ABOUT
    TOP --> WORKS
    TOP --> CERT
    TOP --> CONTACT

    WORKS --> GAMES
    WORKS --> UI

    GAMES --> GAME_DETAIL
    UI --> UI_DETAIL

    CERT --> CERT_LIST
    CERT --> CERT_CONTENTS
    CERT_CONTENTS --> CERT_DETAIL
```

---

## グローバルナビゲーション

グローバルナビゲーションから任意のページへ直接遷移可能です。

```mermaid
flowchart LR
    NAV["🔗 グローバルナビ"]
    TOP["トップページ (/)"]
    ABOUT["プロフィール (/about)"]
    WORKS["作品一覧 (/works)"]
    CERT["資格・学習コンテンツ (/certifications)"]
    CONTACT["お問い合わせ (/contact)"]

    NAV --> TOP
    NAV --> ABOUT
    NAV --> WORKS
    NAV --> CERT
    NAV --> CONTACT
```

---

## 備考・検討事項

- トップページには各セクションへのバナー・カードリンクを設ける
- 詳細ページには「一覧へ戻る」リンクを設ける
- 全ページにグローバルナビゲーションとフッターを配置する
