# サイトマップ案

ポートフォリオサイトのサイトマップ叩き台です。

---

## サイトマップ

```
/ (トップページ)
├── /about (プロフィール)
│
├── /works (作品一覧)
│   ├── /works/games (ゲーム)
│   │   └── /works/games/:id (ゲーム詳細)
│   └── /works/ui (UI)
│       └── /works/ui/:id (UI詳細)
│
├── /certifications (資格・学習コンテンツ)
│   ├── /certifications/list (取得資格一覧)
│   └── /certifications/contents (学習コンテンツ一覧)
│       └── /certifications/contents/:id (学習コンテンツ詳細)
│
└── /contact (お問い合わせ)
```

---

## ページ概要

| パス | ページ名 | 概要 |
|------|----------|------|
| `/` | トップページ | サイト全体の紹介・各セクションへの導線 |
| `/about` | プロフィール | 自己紹介、スキルセット、経歴など |
| `/works` | 作品一覧 | ゲーム・UI 作品のカテゴリ選択ページ |
| `/works/games` | ゲーム一覧 | 制作したゲームの一覧表示 |
| `/works/games/:id` | ゲーム詳細 | 各ゲームの詳細（説明、使用技術、リンクなど） |
| `/works/ui` | UI一覧 | 制作したUIの一覧表示 |
| `/works/ui/:id` | UI詳細 | 各UIの詳細（説明、使用技術、スクリーンショットなど） |
| `/certifications` | 資格・学習コンテンツ | 資格と学習コンテンツのカテゴリ選択ページ |
| `/certifications/list` | 取得資格一覧 | 取得した資格の一覧・バッジ表示 |
| `/certifications/contents` | 学習コンテンツ一覧 | 資格学習用コンテンツ（練習問題など）の一覧 |
| `/certifications/contents/:id` | 学習コンテンツ詳細 | 各コンテンツの詳細・利用ページ |
| `/contact` | お問い合わせ | 問い合わせフォームまたは連絡先の案内 |

---

## 備考・検討事項

- `:id` の形式は、スラッグ（英数字のタイトル）またはIDで管理する
- トップページに各セクションのハイライトを表示する構成も検討する
- `/certifications/contents/:id` は外部リンク（別リポジトリ）への誘導とする可能性あり
